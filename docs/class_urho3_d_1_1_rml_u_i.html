◆ LoadDocument()

Rml::ElementDocument * Urho3D::RmlUI::LoadDocument ( const ea::string &  path,
void *  placeholderContext = nullptr 
)

Load a specified rml document. When resource reloader is active, returned pointer will be invalidated when associated resource change triggers reloading a document. In such cases it is important to subscribe to OnDocumentReloaded signal and update handle change of document pointer.

Parameters
pathResource path to the .rml file.
placeholderContextOptional context pointer for placeholder substitution (e.g., RmlUIComponent instance).

Load a font resource for RmlUi to use. Set fallback parameter to true if font should be used as a fallback font for unsupported characters.
