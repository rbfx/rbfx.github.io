bool Urho3D::PrefabReference::TryCreateInplace ( )

Try to create instance without spawning any new nodes or components. It may cause some nodes or components to remain if prefab is different. So, this method is not completely reliable, but it's the best we can do. It's the Editor only issue anyway.
